It is a journey of learning the course The Complete React Developer Course (w/ Hooks and Redux).

The course is based on react 16.0, but today the react is 18.2 right now. A lot of things have been changed, so often I have to investigate the new implementation for each point of the course.

Below are the differences to the course:

  • Using React 18.2, so Redux, Router, etc are changed
  • Using React Testing Library, because Enzyme is deprecated in React 18
  • Using Webpack 5, so change to use mini-css-extract-plugin and html-webpack-plugin
  • Using Azure, because Heroku is not free right now
  • DoNot using express, because Github provides pipeline to deploy to Azure
